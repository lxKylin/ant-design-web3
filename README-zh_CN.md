## 特性

- 🎨 友好的 Ant Design 设计风格
- 📦 开箱即用的体验
- 🔌 兼容不同的链

## 安装

```shell
npm i @ant-design/web3 --save
```

## 使用

```ts
import { Address } from '@ant-design/web3';

export default () => {
  return <Address address="0x1234567890123456789012345678901234567890" />;
};
```

## 开发

```shell
pnpm i
pnpm dev
```
